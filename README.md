- R2: Keyes O, Hutson J, & Durbin M, A Mulching Proposal
- R3: Wickham H, Tidy Data
- R3: Woo K & Broman K, Data in Spreadsheets

- R5: Angwin J, Larson J, Mattu S & Kirchner L, Machine Bias
